# Coders of Lucknow 🚀

## 📌 Description
This is a Python-based project where I analyzed data of coders from Lucknow using core Python programming.  
The project focuses on understanding user relationships and building simple recommendation features without using external libraries.

---

## 🎯 Objective
- Analyze coder data using core Python  
- Identify relationships between users  
- Build recommendation logic  
- Strengthen problem-solving skills  

---

## 📊 Project Highlights
- Analysis of users with similar interests  
- Identifying relationships between users who liked the same pages  
- Implemented **"People You May Know"** feature  
- Implemented **"Pages You Might Like"** feature  

---

## ⚙️ Technologies Used
- Python (Core Python only)  
- Jupyter Notebook  

---

## 📂 Project Files
- `.ipynb` → Jupyter notebooks  
- `.json` → dataset used for analysis  

---

## ▶️ How to Run
1. Install Python  
2. Open Jupyter Notebook  
3. Run the `.ipynb` files  

---

## 🌟 Features
- Pure Python implementation  
- User relationship mapping  
- Recommendation system logic  
- Clean and structured code
